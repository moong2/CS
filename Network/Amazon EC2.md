# EC2란?
- AWS에서 제공하는 **클라우드 컴퓨팅 서비스**
- 각 세계에 구축한 데이터 센터의 서버용 컴퓨터들의 자원을 원격으로 사용
- AWS에서 제공하는 URL을 통해 임대한 컴퓨터에 접근

# 장점
- **탄력성** : 용량을 늘리거나 줄일 수 있다.
- 사용한만큼 지불한다.
- 사용자가 인스턴스를 완전히 제어
- 보안 및 네트워크 구성, 스토리지 관리가 효과적

# 구성
### 인스턴스
- CPU / RAM

### EBS
- 하드디스크

### ENI
- 랜카드
- 한 네트워크 안에서 컴퓨터간 신호를 주고받는데 쓰임
