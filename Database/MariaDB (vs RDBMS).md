# MySQL
- Oracle Corportation에 인수되어 상업적 지원을 받는 제품
- Oracle 인수 이후 우려 커짐

# MariaDB
- Oracle에 의한 MySQL 인수 이후 시작한 프로젝트
- MySQL의 포크
- MySQL과의 완전한 호환성을 목표로 하지만, 독자적인 기능과 개선 사항을 추가함
- 오픈 소스 커뮤니티 중심으로 개발
- GPL 라이선스 하에 배포

## 성능 최적화
- 복잡한 쿼리 최적화, 서브 쿼리 최적화, 스레드 풀링과 같은 성능 최적화

## 새로운 기능
- JSON 함수 -> MySQL도 존재
- WITH 문
- 윈도우 함수 등

## 기능
- SQL
- NoSQL -> MySQL은 SQL기반이지만 가능하긴 함
