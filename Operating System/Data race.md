# Race Condition이란?
두 개 이상의 스레드 또는 프로세스가 공유 자원에 대해 동시에 읽기 및 쓰기 작업을 수행하려고 할 때, 그 순서에 따라 시스템의 동작이 달라지는 상황
스레드들 간의 실행 순서가 결과에 영향을 미침
### 발생 조건
- 데이터 레이스
- 잘못된 타이밍에 인터러브가 발생
# Data Race란?
두 개 이상의 스레드 또는 프로세스가 동시에 같은 메모리 위치에 쓰기 작업을 하는 것
Race Condition의 부분집합 (포함될 수도, 안 될 수도)
